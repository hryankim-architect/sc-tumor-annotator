# v0.2 — subclonal-CNV regime + CNV-feature ablation

**Type:** capability deepening (no change to the honest-scope framing)

## What v0.2 adds

v0.1 showed the four-stage annotator runs end-to-end. v0.2 makes the central
claim — that an expression-derived CNV signal carries malignant-cell
information — **measurable**, via a hard regime and an ablation.

- `synth.generate_malignancy_cohort()` — a hard regime where normal and
  malignant epithelial cells share an *identical* transcriptomic baseline and
  differ only by **heterogeneous, subclonal CNV**: each malignant cell alters a
  random subset of chromosomes with a random sign, at small per-gene amplitude
  buried under noise. No non-CNV marker separates the two classes, and because
  the alteration sign varies cell to cell there is no single linear separating
  axis — so a magnitude-based CNV score is the natural signal.
- `ablation.cnv_ablation()` — cross-validated malignant-call macro-F1 across
  three feature sets (embedding / CNV-scalar-only / embedding + CNV) plus a kNN
  reference-mapping baseline.
- `adapter.from_anndata()` — optional real-data bridge (Scanpy/AnnData +
  gene→chromosome map) so the pipeline can run on real cohorts.
- `notebooks/demo.ipynb` — CNV score density, inferred-CNV heatmap, and the
  ablation bar chart.

## Ablation result (synthetic hard regime, seed 0; malignant-call macro-F1)

| Feature set | macro-F1 |
|---|---|
| kNN reference-mapping (embedding) | 0.92 |
| **CNV scalar only (1 feature)** | **0.943** |
| Embedding (30 PCs) | 0.986 |
| Embedding + CNV | 0.990 |

Honest reading: a **single, biologically-grounded CNV scalar recovers the
malignant call at 0.94 macro-F1** — within ~4 points of a 30-dimensional
transcriptomic embedding — and adding it to the embedding is non-harmful and
slightly additive (0.986 → 0.990, +0.0035). A gradient-boosted tree can already
recover much of the CNV magnitude from the embedding nonlinearly, so the value
of the explicit CNV score is **interpretability and compactness**, not a large
accuracy jump. The kNN baseline trails because the sign-heterogeneous
alterations give it no single linear axis to map along.

## Honest scope

Still synthetic-only; the numbers describe the demo regime, not a real-cohort
benchmark. See `what-is-out-of-scope.md`.
